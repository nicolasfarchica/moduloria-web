## Tono de Voz

### Personalidad de Marca

- **Profesional** pero accesible
- **Experto** sin ser arrogante
- **Directo** sin ser agresivo
- **Práctico** sin ser aburrido

### Palabras a USAR

| Categoría | Términos |
|-----------|----------|
| Resultados | medible, tangible, visible, real, concreto |
| Tiempo | rápido, ágil, en semanas, inmediato |
| Confianza | sin compromiso, sin ataduras, transparente |
| Expertise | especializado, enfocado, dedicado |
| Valor | ROI, eficiencia, optimización, automatización |

### Palabras a EVITAR

| Categoría | Términos |
|-----------|----------|
| Números específicos | "40% ahorro", "50% menos tiempo", "3x más rápido" |
| Precio | gratis, gratuito, sin costo, económico |
| Exageración | revolucionario, disruptivo, mágico, increíble |
| Futurista | robots, el futuro es ahora, ciencia ficción |

### Frases Aprobadas

```
✓ "Resultados medibles en semanas"
✓ "ROI visible desde el primer piloto"
✓ "Ahorro significativo de tiempo"
✓ "Sin compromisos de permanencia"
✓ "Diagnóstico Estratégico"
✓ "Evaluación inicial sin compromiso"
✓ "Primera conversación"
```

### Frases NO Aprobadas

```
✗ "Ahorra un 40% de tiempo"
✗ "Diagnóstico gratuito"
✗ "Auditoría gratis"
✗ "El futuro de la construcción"
✗ "Robots para tu empresa"
✗ "Transformación digital revolucionaria"
```

## Estructura de Posts

### Carrusel Educativo (8 slides)

1. **Portada**: Hook + título atractivo
2. **Problema**: Pregunta o dolor identificable
3. **Agitación**: Consecuencias del problema
4. **Solución**: Qué ofrecemos
5. **Cómo funciona**: Proceso o metodología
6. **Beneficios**: 3-4 puntos clave
7. **Prueba/Autoridad**: Credibilidad
8. **CTA**: Llamada a la acción clara

### Post de Texto (LinkedIn)

```
[Hook - primera línea impactante]

[Desarrollo del problema - 2-3 párrafos]

[Solución propuesta - puntos claros]

[CTA + pregunta de engagement]

[Hashtags - máximo 5-7]
```

## Templates Canva Recomendados

### Para carruseles
- Buscar: "Business Carousel" o "Corporate Carousel"
- Personalizar con colores de marca
- Mantener consistencia entre slides

### Para posts individuales
- Buscar: "Professional Social Media"
- Evitar templates demasiado coloridos
- Preferir diseños limpios y minimalistas

## Checklist de Publicación

### Antes de crear

- [ ] Revisar calendario editorial
- [ ] Definir objetivo del post
- [ ] Identificar plataforma principal

### Durante el diseño

- [ ] Usar colores de marca correctos
- [ ] Tipografía Inter (Bold/Regular)
- [ ] Logo visible y bien posicionado
- [ ] Dimensiones correctas para la plataforma
- [ ] Sin números específicos de ahorro
- [ ] Sin palabra "gratis/gratuito"

### Antes de publicar

- [ ] Copy revisado (sin errores ortográficos)
- [ ] Hashtags preparados
- [ ] Link verificado (moduloria.com/auditoria)
- [ ] Horario óptimo seleccionado
- [ ] Primer comentario/respuestas preparadas

### Después de publicar

- [ ] Responder comentarios en primera hora
- [ ] Monitorear métricas
- [ ] Documentar aprendizajes
