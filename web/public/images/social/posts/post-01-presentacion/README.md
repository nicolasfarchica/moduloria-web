## Cómo crear el carrusel en Canva

1. Abrir Canva y crear diseño personalizado 1080x1350px
2. Configurar colores de marca:
   - Azul oscuro: #1a1f36
   - Dorado: #d4a574
3. Agregar tipografía Inter (disponible en Canva)
4. Seguir especificaciones de `ESPECIFICACIONES-CARRUSEL.md`
5. Exportar cada slide como PNG individual
6. Nombrar archivos: `slide-01-portada.png`, `slide-02-problema.png`, etc.
