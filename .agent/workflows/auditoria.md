---
description: Auditoría del estado actual del proyecto para recuperar contexto
---

Usa este workflow cuando te sientas perdido o sientas que el proyecto se está desorganizando.

1. **Mapa de Estructura**:
   - Lista los directorios principales (`web/src/components`, `web/src/app`).
   - Identifica componentes "huérfanos" o archivos que no se usan.

2. **Revisión de CLAUDE.md**:
   - Verifica si la documentación en `CLAUDE.md` sigue siendo válida.
   - Si se han añadido nuevas reglas de negocio, actualízalas.

3. **Check de Consistencia**:
   - ¿Todos los componentes siguen el mismo estilo visual (Nordic Light/Electric AI)?
   - ¿Las variables de Tailwind o CSS están centralizadas?

4. **Informe de Deuda Técnica**:
   - Haz una lista de "TODOs" pendientes que podrían causar problemas a futuro.
   - Prioriza qué arreglar primero.
