## Aprendizajes Importantes

### Formatos por Plataforma (MEMORIZAR)

```
LinkedIn / Facebook → 4:5 vertical → Gamma: "dimensions": "4x5"
Instagram → 1:1 cuadrado → Gamma: "dimensions": "1x1"
```

**LinkedIn y Facebook pueden compartir el mismo archivo.**
**Instagram SIEMPRE necesita version cuadrada separada.**

### Parametro para Logo en Slides

```json
"cardOptions": {
  "headerFooter": {
    "bottomRight": {
      "type": "image",
      "source": "themeLogo",
      "size": "sm"
    }
  }
}
```
