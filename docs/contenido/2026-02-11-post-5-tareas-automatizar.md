#### Slide 3: TAREA #1
```
## 1. Seguimiento de facturas de proveedores

Hoy: Tu equipo dedica 8-12 horas/mes a recibir facturas
por email, copiar datos a mano en Excel, cruzar con pedidos
y perseguir discrepancias. Errores de transcripcion,
pagos duplicados, retrasos.

Con IA: Un sistema OCR lee la factura, extrae los datos,
los cruza automaticamente con los pedidos y te alerta
solo cuando hay discrepancias.

Resultado: 30 minutos/mes. Cero errores.
```

#### Slide 4: TAREA #2
```
## 2. Respuestas a consultas repetitivas

Hoy: Tu equipo recibe 100-200 consultas/mes sobre lo mismo:
precios base, plazos de entrega, documentacion tecnica,
disponibilidad. Cada respuesta lleva 5-10 minutos
y interrumpe el trabajo real.

Con IA: Un chatbot entrenado con tu informacion responde
el 70% de estas consultas automaticamente, 24/7,
sin que nadie tenga que parar lo que esta haciendo.

Resultado: 20 horas/mes ahorradas = €1,200/mes
```

#### Slide 5: TAREA #3
```
## 3. Generacion de presupuestos

Hoy: Cada presupuesto nuevo empieza buscando uno viejo
en Excel, copiando, pegando, actualizando precios a mano.
3-4 horas por presupuesto. Inconsistencias entre versiones.
El cliente espera dias.

Con IA: Introduces los parametros del proyecto y el sistema
genera un presupuesto estandarizado en minutos, con precios
actualizados y formato profesional.

Resultado: De 3-4 horas a 15 minutos por presupuesto.
```

#### Slide 6: TAREA #4
```
## 4. Recordatorios y seguimiento de proyectos

Hoy: Alguien tiene que acordarse de enviar emails
para cada hito, cada vencimiento, cada revision pendiente.
Si se olvida, se pierde tiempo, se incumplen plazos
y el cliente se entera tarde.

Con IA: El sistema detecta fechas clave y envia
notificaciones automaticas por email, WhatsApp o SMS
al equipo correcto, en el momento correcto.

Resultado: 10 horas/mes ahorradas. Cero olvidos.
```

#### Slide 7: TAREA #5
```
## 5. Reportes de obra semanales

Hoy: El jefe de obra pasa 4-6 horas cada viernes
juntando fotos del movil, notas en papel, datos de avance.
Copia todo en un Word. Lo envia por email.
El lunes ya esta desactualizado.

Con IA: El equipo sube fotos y notas desde una app.
El sistema genera el reporte automaticamente con fotos
organizadas, metricas de avance y comparativa con plan.

Resultado: 20 horas/mes ahorradas. Reportes siempre al dia.
```
