# Sesion 22 Enero 2026: Sistema de Redes Sociales

## Resumen

Creacion del sistema completo de gestion de redes sociales sostenible para ModulorIA.

## Problema a Resolver

Nicolas no disfruta las redes sociales y le cuesta mantener consistencia. Necesita un sistema que:
- Minimice el tiempo invertido (3-4 horas/semana max)
- Mantenga calidad y autoridad
- Sea sostenible a largo plazo
- Use herramientas accesibles (~50 EUR/mes)

## Solucion Implementada

**Filosofia:** "Crear 1 vez, distribuir 4 veces"

### Stack Confirmado
- **Metricool Plan Start** - Programacion y analytics (pago 23 enero)
- **Gamma** - Carruseles (ya tiene)
- **FreePik** - Imagenes (ya tiene)
- **Claude** - Ideacion y borradores (ya tiene)

### Flujo de Trabajo
Sesion de batching cada 2 semanas:
1. Ideacion (30 min)
2. Redaccion (1.5h)
3. Visuales (1.5h)
4. Programacion (30 min)

**Total:** 4 horas cada 2 semanas = contenido para 2 semanas
