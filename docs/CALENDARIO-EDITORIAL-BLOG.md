## RECORDATORIO SEMANAL (Sistema de Alertas)

### Opcion 1: Recordatorio Manual (Calendario)
Crear evento recurrente en Google Calendar:
- **Titulo**: "Blog ModulorIA - Preparar Articulo"
- **Dia**: Cada LUNES a las 9:00 AM
- **Descripcion**: "Preparar y publicar articulo semanal. Ver: docs/CALENDARIO-EDITORIAL-BLOG.md"
- **Recurrencia**: Semanal, indefinida

### Opcion 2: N8N Workflow (Automatizado)
Crear workflow que envie recordatorio por WhatsApp/Email:
- **Trigger**: Cron cada lunes 9:00 AM
- **Accion**: Enviar mensaje "Hoy toca preparar el articulo del blog. Proximo keyword: [X]. Publicar manana martes."
- **Status**: PENDIENTE IMPLEMENTACION

### Checklist Publicacion (Cada Martes)

```
PRE-PUBLICACION (Lunes):
[ ] Revisar outline del articulo en este calendario
[ ] Escribir articulo (1,500-2,000 palabras)
[ ] Verificar internal links a otros articulos
[ ] Crear hero image (1200x670px, JPEG, <300KB)
[ ] Frontmatter completo (title, date, excerpt, tags, category, image)

PUBLICACION (Martes):
[ ] git add src/content/blog/[slug].md public/images/blog/hero-[slug].jpg

[ ] git push origin main
[ ] Verificar en https://moduloria.com/blog/[slug] (2-3 min)
[ ] Submit URL en Google Search Console (Inspeccionar URL -> Solicitar indexacion)

POST-PUBLICACION (Martes-Miercoles):
[ ] Compartir link en LinkedIn personal
[ ] Incluir en newsletter semanal (si aplica)
[ ] Actualizar PILLAR con link al nuevo articulo (si es cluster)
```

## VERIFICACION SEO POR ARTICULO

Cada articulo publicado debe tener:

| Elemento | Donde | Verificacion |
|----------|-------|-------------|
| Title tag | Frontmatter `title` | <60 chars, incluye keyword |
| Meta description | Frontmatter `excerpt` | <160 chars, incluye keyword, CTA implicito |
| H1 | Template (automatico desde title) | Solo 1 por pagina |
| H2s | Contenido markdown | 5-8 por articulo, keywords naturales |
| Keyword density | Contenido | 1-2% keyword principal |
| Internal links | Contenido | 3-5 links a otros articulos |
| External links | Contenido (si aplica) | 1-2 fuentes autoritativas |
| Image alt | Frontmatter `title` (automatico) | Descriptivo, incluye keyword |
| URL/Slug | Nombre archivo .md | Keyword principal en slug |
| Canonical | Template (automatico) | moduloria.com/blog/[slug] |
| OG Image | Frontmatter `image` | 1200x670px JPEG |
| Structured Data | Template (JSON-LD Article) | Schema.org/Article |
| Reading time | Calculado automatico | Se muestra en header |
| Tags | Frontmatter `tags` | 3-6 tags relevantes |
