## Horarios Optimos por Plataforma

### LinkedIn
- **Mejor:** 8:00 - 10:00 (Martes a Jueves)
- **Bueno:** 12:00 - 14:00 (cualquier dia laboral)
- **Evitar:** Despues de las 18:00, fines de semana

### Facebook
- **Mejor:** 9:00 - 12:00 (Miercoles)
- **Bueno:** 13:00 - 16:00 (Jueves, Viernes)
- **Fines de semana:** Funciona bien para B2C, menos para B2B

### Instagram
- **Mejor:** 11:00 - 13:00, 19:00 - 21:00
- **Bueno:** Cualquier dia entre 9:00 - 21:00
- **Nota:** Historias pueden ir a cualquier hora

## Reglas del Calendario

### Hacer:
- Alternar entre pilares (no 3 educativos seguidos)
- Carruseles a principio de semana (mas engagement)
- Mantener consistencia de dias/horas
- Dejar huecos para contenido espontaneo

### No hacer:
- Publicar mas de 1 vez al dia en LinkedIn
- Posts de venta directa mas de 1 vez cada 2 semanas
- Publicar fines de semana contenido B2B importante
- Cambiar horarios constantemente

```
LinkedIn Personal (contenido original)
         |
         v
    [Auto-share]
         |
    +----+----+
    |         |
    v         v
LinkedIn   Facebook
Empresa    Empresa
    |         |
    v         v
[Version adaptada para Instagram si aplica]
```

### Adaptaciones por Plataforma

| Original LinkedIn | Facebook | Instagram |
|-------------------|----------|-----------|
| Carrusel PDF | Mismo PDF o imagenes separadas | Carrusel nativo |
| Post texto largo | Version resumida + link | Version muy corta + imagen |
| Opinion/reflexion | Mismo texto | Skip o story |

## Contenido Curado (Paginas Empresa)

Para llenar gaps sin crear contenido nuevo:

### Fuentes recomendadas:
- Noticias sector construccion
- Estudios/reportes industria
- Contenido de partners/proveedores
- Articulos relevantes sobre IA

### Template post curado:

## Notas

- Este calendario es una guia, no una obligacion
- Ajustar segun lo que funcione (revisar metricas)
- Si una semana es imposible, 2 posts es mejor que 0
- Calidad > cantidad siempre
